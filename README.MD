# Narwhal

A .NET Core CLI to save and load docker volumes to tarballs

# Getting Started

Install via DotNet CLI
```bash
$ dotnet tool install -global narwhal
```

Check help command
```bash
$ narwhal
```
or
```bash
$ narwhal --help
```
